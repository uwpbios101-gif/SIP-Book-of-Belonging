1. Create or open the GitHub repository folder.
2. Copy every file and folder from this package into the repository root.
3. Confirm index.html is directly in the root.
4. Commit: Launch the SIP Book of Belonging
5. Push origin.
6. In GitHub: Settings > Pages > Deploy from a branch > main / root.
