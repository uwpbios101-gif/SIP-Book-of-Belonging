# The Book of Belonging — SIP

A responsive occasion-discovery and promotional website for SIP.

## Included
- Nine founding promotional campaigns
- Emotional chapter filtering
- Search
- Moment Finder inquiry form
- Fullscreen mode
- Responsive mobile design
- Offline PWA cache
- Print styles

## Publish
Place all files in the repository root, commit, push, and enable GitHub Pages from the `main` branch root.
